IPython: Productive Interactive Computing¶

IPython provides a rich toolkit to help you make the most out of using Python interactively. Its main components are:

  • Powerful interactive Python shells (terminal- and Qt-based).
  • Support for interactive data visualization and use of GUI toolkits.
  • Flexible, embeddable interpreters to load into your own projects.
  • Tools for high level and interactive parallel computing.

IPython is open source (released under the revised BSD license), and is used by a range of other projects. The talks and presentations we have given about IPython provide a good overview of its capabilities.

IPython supports Python 2.6 to 2.7 and 3.1 to 3.2. Using it with Python 3 requires a separate download, described in our Python 3 wiki page. Our older 0.10 series supports Python 2.5, and probably works well with 2.4.

Citing IPython¶

Several of the authors of IPython are connected with academic and scientific research, so it is important to us to be able to show the impact of our work in other projects and fields.

If IPython contributes to a project that leads to a scientific publication, please acknowledge this fact by citing the project. You can use this ready-made citation entry.
